# 영화관 좌석 리뷰 등록 웹사이트(롯데시네마)

### 주요기능
1. 영화관 선택.
2. 영화 제목, 좌석, 평점, 리뷰 작성
3. 내용을 모두 작성 후 입력버튼을 눌렀을 때 리스트에 추가, 빈칸이 있다면 알림창.
4. 리스트 삭제 버튼 구현

-이것은 스터디그룹 팀원들과 처음 만든 프로젝트입니다.
